WELD TRACK

A Weld Track é uma API feita para fazer a gestão, monitorização e controlo dos soldadores e dos seus registos de trabalho. A aplicação permite registar a atividade diária, os tempos gastos e garantir a validação dos dados.

Tecnologias Usadas

Java 21Versão estável atualizada
SpringBoot 3.4.2Framework principal
PostgreSQLBase de Dados ativa
Springdoc OpenAPI 2.7.0Documentação interativa
MavenGestor de dependências

Pré-Requisitos & Execução

  • Java 21 instalado
  • PostgreSQL ativo (Porta 5432)
  • Maven configurado

Abra o projeto no IntelliJ, aguarde o Maven e execute a classe Main.

Endpoints Disponíveis por Classe

  • GET - Lista a informação
  • GET (ID) - Procura pelo ID
  • POST - Cria um novo registo
  • PUT - Atualiza os dados
  • DELETE - Apaga um registo
